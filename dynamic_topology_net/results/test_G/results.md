# Test G: Pruning Error vs Singular Value

Seed: 42, Epochs: 24, LR: 0.08, Batch: 128, Widths: [8, 16, 32]

For each neuron in a trained diagonalised network: measure L2 output error
and accuracy drop when that single neuron is pruned.

## Correlation Analysis

- Pearson correlation (singular_value, L2_error): 0.7704
- Pearson correlation (singular_value, acc_drop): 0.7115

**Strong positive correlation between SV and L2 error.** Neurons with small
singular values cause significantly less output change when pruned.
This quantitatively supports the paper's 'minimal degradation' claim.
